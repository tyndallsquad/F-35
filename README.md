## Features

-   Stealth multirole fighter concept
-   Air-to-Air (AA) capability
-   AGM-154 support
-   JDAM guided bombs
-   Infrared (IR) targeting
-   Night Vision (NV) operations
-   IFF & Data Link

## Installation

1.  Download or clone this repository into your FlightGear `Aircraft`
    folder.
2.  Launch FlightGear and select **F-35** from the aircraft list.

## Controls

-   Standard FlightGear flight controls
-   Weapon release: configured via simulator weapon bindings
-   IR / NV: toggle
